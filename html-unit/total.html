HTML 简介

  • HTML 指的是超文本标记语言(Hyper Text Markup Language)
  • HTML 不是一种编程语言，而是一种标记语言(Marlup Language)
  • 标记语言是一套标记标签(Markup Tag)
  • HTML 使用标记标签来描述网页

HTML 基础

  • html标题 h1,h2,h3,h4,h5,h6
  • html文本 p
  • html链接 a
  • html图像 src

HTML 元素

  • HTML 元素以开始标签起始
  • HTML 元素以结束标签终止
  • 元素的内容是开始标签和结束标签之间的内容
  • 某些 HTML元素具有空内容(empty content)
  • 空元素在开始标签中进行关闭(以开始标签的结束而结束)
  • 大多数 HTML元素可拥有属性

br hr就属于没有关闭标签的空元素

HTML 属性

  • HTML 标签可以拥有属性。属性提供了有关HTML元素更多的信息
  • 属性总是以名称/值对的形式出现,例如:name="value"
  • 属性总是在HTML元素的开始标签中规定

HTML 标题

  • 标题(heading)是通过<h1>~<h6>等标签进行定义的.
  • <h1>定义最大的标题,<h6>定义最小的标题.
  • 浏览器会自动的在标题的前后添加空行.
  • 默认情况下,HTML会自动的在块级元素前后添加一个额外的空行,比如段落、标题元素前后.
  • 请确保将HTML heading标签只用于标题.不要仅仅是为了产生粗体或大号的文本而使用标题.
  • 搜索引擎使用标题为您的网页结构和内容编制索引
  • 用户可以通过标题来快速浏览网页,所以用标题来呈现文档结构很重要.

HTML 注释

实例:  <!-- This is a comment -->

HTML 段落

  • 浏览器会自动的在段落的前后加上空行(<p>是块级元素)
  • 使用空的<p></p>插入空行是个坏习惯,用<br/>代替它.
  • 当显示页面的时候,浏览器会移除源代码中多余的空行和空格.所有连续的空行和空格都被视作一个空格.

HTML 链接

  • 通过使用href属性 创建一个指向另一个文档的链接
  • 通过使用name(id)属性 创建文档的书签

HTML 图像

  • <img>是空标签，只包含属性，并没有闭合标签
  • 加载图片需要时间，慎用图片

HTML 表格

  • 表格由<table>标签来定义。每个表格均有若干行(由<tr>标签定义),每行被分为若干单元格(由<td>标签定义).
  • 表格的表头有<th>来定义，大多数浏览器会把表头显示为粗体居中的文本。
  • 空格占位符 &nbsp 会解决空单元格显示不友好问题

HTML 列表

  1. 列表分为有序列表、无序列表、定义列表
  2. 有序列表 <ol>element≤/ol>
  3. 无序列表 <ul>element</ul>
  4. 定义列表 <dl /> <dt /> <dd />

HTML 表单

  • 表单是一个包含表单元素的区域
  • 表单元素是允许用户在表单中(比如：文本域、下拉列表、单选框、复选框等)输入信息的元素
  • 表单使用表单标签 <form> 定义
  • 大多数情况下用到的表单标签是输入标签 <input> 输入类型由类型属性 type 定义
输入标签类型 类型名称
文本域 TextFields input type="text"
单选按钮 Radio Buttons input type="radio"
复选框 Checkboxes input type="checkbox"
确认按钮 input type="submit"
按钮 input type="button" value="Hello"

下拉列表: select option value selected

HTML 框架

  • 框架可以同一个浏览器窗口显示多个页面。每份html文档称为一个框架，每份框架都独立于其他框架
  • 框架结构标签 <frameset>
  • 框架标签 <frame>
  • 注意：不能将<body></body>标签与<frameset></frameset>标签同事使用。不过，假如添加一段文本的<noresize>标签，就必须将这段文字嵌套于<body></body>标签内

内联框架

  1. iframe 用于在网页内显示网页
  2. height width 属性用于规定iframe的高度和宽度
  3. frameborder 属性规定iframe周围的边框
  4. iframe 可以用作链接的目标(target)，链接的target属性必须引用iframe的name属性

HTML 头部

  1. <head>元素是所有头部元素的容器，<head>内的元素可以包含脚本，指示浏览器在何处可以找到样式表，提供元素信息，等等
  2. <title> <base> <link> <meta> <script> <style> 都可以添加到head部分
title 元素
<title>定义文档的标题
  • 定义浏览器中工具栏的标题
  • 提供页面被添加到收藏夹时显示的标题
  • 显示在搜索引擎结果中的页面标题
base 元素
<base> 标签为页面上的所有的链接规定默认地址和默认目标(target)
link 元素
<link> 标签定义文档与外部资源之间的关系
<link> 标签最常用的是链接样式表
style 元素
<style> 标签用于为HTML文档定义样式信息
meta 元素
  1. 元数据(metadata)是关于数据的信息
  2. <meta>标签提供关于HTML文档的元数据。元数据不会显示在页面上，但是对于机器是可读的。
  3. <meta>标签始终被用于head元素中
